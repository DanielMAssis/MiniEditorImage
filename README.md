# MiniEditorImage
Teste a demo [aqui](https://danielmassis.github.io/MiniEditorImage/).

# Descrição
Editor de imagem simples com funções de cortar e aplicar filtros nas imagens.

## Funções
* Cortar
* Blur
* Brilho
* Contraste
* Gray Scale
* Matiz
* Inverter Cores
* Opacidade
* Saturação
* Sépia
